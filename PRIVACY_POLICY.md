## 日本語版

### データ収集について

**Webflow管理画面日本語化** Chrome拡張機能（以下「本拡張機能」）は、ユーザーのプライバシーを最優先に考えて設計されています。

本拡張機能は以下を保証します：

#### 収集しないデータ

- ❌ 個人情報（名前、メールアドレス、電話番号等）
- ❌ 閲覧履歴
- ❌ Webflowプロジェクトの内容
- ❌ フォーム入力内容
- ❌ Cookie情報
- ❌ ログイン情報
- ❌ その他いかなるユーザーデータ

#### データの送信

- ❌ 外部サーバーへのデータ送信を一切行いません
- ✅ すべての翻訳処理はブラウザ内（ローカル）で完結します
- ✅ インターネット接続は必要ありません（インストール後）

#### 使用する権限

本拡張機能が要求する権限とその理由：

1. **storage権限**
   - 目的: 将来的なユーザー設定の保存用
   - 現在の使用状況: 未使用（将来のバージョンで利用予定）
   - データの保存先: ブラウザのローカルストレージのみ

2. **host_permissions: https://*.webflow.com/***
   - 目的: Webflowサイトのページ内容を翻訳するため
   - 動作範囲: Webflowドメイン（*.webflow.com）のみ
   - その他のウェブサイトには一切アクセスしません

#### 動作の仕組み

1. ユーザーがWebflow（*.webflow.com）にアクセス
2. 拡張機能が同梱されたCSVファイル（translation_terms.csv）を読み込み
3. ページ内の英語テキストを日本語に置換（ブラウザ内で完結）
4. 外部通信は一切発生しません

#### サードパーティサービス

- 本拡張機能は、Google Analytics等のトラッキングツールを使用していません
- 広告配信サービスを使用していません
- いかなるサードパーティサービスともデータを共有しません

#### オープンソース

本拡張機能はオープンソースであり、すべてのソースコードが公開されています。

- GitHubリポジトリ: [https://github.com/yourusername/webflow-ja-extension](https://github.com/yourusername/webflow-ja-extension)
- コードレビューと監査が可能です

#### プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。
重要な変更がある場合は、拡張機能のアップデート情報でお知らせします。

## English Version

### Data Collection

**Webflow Japanese Translation** Chrome Extension (the "Extension") is designed with user privacy as the top priority.

#### Data We Do NOT Collect

- ❌ Personal information (name, email, phone number, etc.)
- ❌ Browsing history
- ❌ Webflow project content
- ❌ Form input data
- ❌ Cookie information
- ❌ Login credentials
- ❌ Any other user data

#### Data Transmission

- ❌ No data is transmitted to external servers
- ✅ All translation processing is performed locally in the browser
- ✅ Internet connection is not required (after installation)

#### Permissions Used

Permissions requested by this extension and their purposes:

1. **storage permission**
   - Purpose: For future user settings storage
   - Current usage: Not used (planned for future versions)
   - Storage location: Browser's local storage only

2. **host_permissions: https://*.webflow.com/***
   - Purpose: To translate page content on Webflow sites
   - Scope: Webflow domain (*.webflow.com) only
   - No access to any other websites

#### How It Works

1. User visits Webflow (*.webflow.com)
2. Extension loads bundled CSV file (translation_terms.csv)
3. Replaces English text with Japanese on the page (entirely in-browser)
4. No external communication occurs

#### Third-Party Services

- This extension does not use Google Analytics or other tracking tools
- No advertising services are used
- No data is shared with any third-party services
