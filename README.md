# OCディア管理画面 日本語化 Chrome拡張機能

## 概要
OCディアの管理画面の英語UIを、CSVで管理する用語集に基づき日本語化するChrome拡張機能です。DOM監視（MutationObserver）で動的要素もカバーします。用語が未登録の場合は英語表示のまま（フォールバック）になります。

- 英語UIテキストを部分的に日本語へ自動置換
- 用語集（CSV）は自由に編集・拡張可能
- オプションページで用語集一覧・部分置換プレビューが可能
- 許可ホスト/URLパターンを設定し、対象サイトだけに適用

## インストール方法
1. このリポジトリをダウンロードまたはクローンします。
2. Chromeの `chrome://extensions/` を開き、「デベロッパーモード」をONにします。
3. 「パッケージ化されていない拡張機能を読み込む」から本フォルダを選択します。
4. OCディアの管理画面を開き、日本語化が反映されているかご確認ください。

## 使い方
- OCディアの管理画面にアクセスすると、自動で日本語化が適用されます。
- オプションページ（拡張機能の「詳細」→「オプション」）で、用語集の一覧や部分置換プレビューが利用できます。
- 許可ホスト/URLパターンを設定して、対象サイト（例: `*.ocdia.com`）のみに適用できます。

## 翻訳用語集（CSV）の更新方法
- `translation_terms.csv` を直接編集し、英語-日本語のペアを追加・修正してください。
- 編集後、拡張機能を「再読み込み」すると新しい翻訳が反映されます。
- Googleスプレッドシート等で管理→CSVエクスポート→上書きも可能です。
